## Overview

The Jazz Groove Analyzer (JGA) is an original research project dedicated to the computational analysis of timing, groove and temporal interaction in jazz ensembles.

The project reconstructs the **Timeline Metrica Interna (TMI / Internal Metric Timeline - IMT)** directly from audio recordings.
