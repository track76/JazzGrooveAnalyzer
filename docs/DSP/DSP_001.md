# DSP-001

# DSP Philosophy

## Purpose

This document establishes the Digital Signal Processing philosophy adopted by the Jazz Groove Analyzer.

DSP is regarded exclusively as the computational implementation of the mathematical model previously defined.

DSP shall never introduce new theoretical concepts.

It shall only provide computational procedures capable of constructing the mathematical objects defined by the Mathematical Foundations.

## Principles

- Theory precedes DSP.
- Mathematics precedes algorithms.
- DSP implements mathematics.
- Different DSP algorithms may implement the same mathematical model.
- DSP never modifies the theoretical framework.
