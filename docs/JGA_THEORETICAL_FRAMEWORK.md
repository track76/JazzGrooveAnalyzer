# 1. Introduction

The Jazz Groove Analyzer (JGA) is a scientific framework for the mathematical
representation of collective metric behaviour in musical ensembles.

Rather than estimating beats or tempo directly from audio, the JGA constructs
a hierarchy of progressively higher-level representations describing the
observable temporal organisation emerging from ensemble interaction.

The software implements the scientific theory.
The scientific theory always precedes the software.

---

# 2. Scientific Problem

The objective of the JGA is to study how a collective metric reference
emerges from observable rhythmic events generated by multiple interacting
musical sources.

The framework does not attempt to infer performer intentions,
expressive meaning or aesthetic qualities.

Only observable temporal properties belong to the scientific model.

---

# 3. Observed Phenomenon

The phenomenon investigated by the JGA is:

> Collective Metric Behaviour

Collective Metric Behaviour is the observable temporal organisation emerging
from the interaction of multiple rhythmic contributors within an ensemble.

The framework progressively reconstructs this behaviour through successive
representational levels.

---

# 4. Fundamental Principles

The following principles are invariant.

1. Theory always precedes implementation.
2. Every representation has a precise scientific definition.
3. Every Domain object corresponds to one theoretical object.
4. Every transformation is deterministic.
5. Every transformation has formally defined Input and Output.
6. No implicit transformations are permitted.
7. Every representation depends exclusively on the immediately preceding one.
8. Behaviour is represented, never inferred.
9. Only observable properties belong to the scientific model.
10. The Core never assigns musical meaning.
11. The Domain interprets only validated representations.
12. Scientific reproducibility has priority over implementation convenience.

---

---

# 6. Observation Layer

The Observation Layer contains the observable temporal information extracted
from the audio signal.

It introduces no musical interpretation.

Its responsibility is exclusively the production of observable temporal
representations.

---

# 7. Metric Context

Metric Context represents the validated observable temporal structure
produced by the Observation Layer.

It constitutes the complete observational description required for Domain
translation.

Metric Context contains no musical interpretation.

---

# 8. Translation Layer

The Translation Layer performs the deterministic transformation between the
validated Observation Layer and the Domain Layer.

Translation preserves all observable information.

No information is invented.

No information is discarded without explicit theoretical justification.

---

# 9. Elementary Metric Event

The Elementary Metric Event (EME) is the smallest Domain representation of
an observable metric event.

Each EME preserves the observable evidence generated by the Translation Layer.

---

# 10. Beat Reference

Beat Reference represents the theoretical temporal reference reconstructed
from collective metric evidence.

It is never directly observed.

---

# 11. Metric Cluster

A Metric Cluster groups the Elementary Metric Events supporting one Beat
Reference.

Metric Clusters preserve the observable evidence that generated the
reconstructed beat.

---

# 12. Pulse

A Pulse represents one reconstructed ensemble pulsation.

Each Pulse derives from exactly one Metric Cluster.

---

# 13. Internal Metric Timeline

The Internal Metric Timeline is the ordered sequence of reconstructed Pulses.

It represents the reconstructed temporal evolution of the ensemble.

It is the highest-level temporal representation.

---

# 14. Behaviour Observation

Behaviour Observation represents the observable temporal behaviour emerging
from the Internal Metric Timeline.

It does not introduce measurements.

It simply represents the observable behaviour of the reconstructed temporal
system.

---

# 15. Behaviour Profile

Behaviour Profile represents the collection of Behaviour Observations
describing a complete musical performance.

It constitutes the highest behavioural representation currently defined by
the JGA.

No quantitative descriptors are introduced at this level.

---

# 16. Scientific Consequences

The proposed framework implies that:

- Beat References emerge from collective rhythmic evidence.
- Metric Clusters preserve observable evidence.
- Pulses are reconstructed theoretical entities.
- Internal Metric Timelines reconstruct temporal continuity.
- Behaviour Observations represent the temporal behaviour of the reconstructed system.
- Behaviour Profiles organise Behaviour Observations without introducing measurements.
- Quantification belongs to a subsequent scientific stage.

---

# 17. Future Developments

The next stage of the framework introduces Behaviour Quantification.

Behaviour Quantification will define deterministic Behaviour Descriptors
derived exclusively from Behaviour Observations.

Descriptors shall never modify previously reconstructed representations.
