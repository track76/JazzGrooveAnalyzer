# F-002 — Observable Musical Facts

Every Elementary Metric Event (EME) is treated as a
scientific representation of an observable musical fact.

The original observed event is never modified.

The EME preserves the temporal information derived from
the Observation Layer.

Each EME may be described through observable attributes:

- timestamp
- source
- classification
- metric position
- metric function
- confidence

The JGA documents musical reality without altering it.
