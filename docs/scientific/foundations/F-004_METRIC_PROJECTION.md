# F-004 — Metric Projection

## Purpose

Metric Projection is the scientific process that maps observable metric
events into the inferred metric representation of the ensemble.

It constitutes the transition from physical time to musical time.

Projection never alters the observed signal.

---

## Inputs

Metric Projection operates on the sequence of Pulse entities obtained
from the previous stages of the Metric Domain.

Every Pulse already represents an inferred metric occurrence derived
from observable musical events.

---

## Outputs

The output of Metric Projection is a Pulse sequence interpreted as the
Internal Metric Timeline.

Projection does not create new observations.

Instead, it assigns metric meaning to the existing Pulse sequence.

---

## Scientific Invariants

Metric Projection shall always satisfy the following invariants.

### Invariant 1

Projection never modifies physical timestamps.

### Invariant 2

Projection never modifies Pulse identity.

### Invariant 3

Projection never changes the chronological ordering of the sequence.

### Invariant 4

Projection introduces no arbitrary musical interpretation.

Every inferred property must originate from observable information
already reconstructed by previous stages.

### Invariant 5

Projection is deterministic.

Given the same Pulse sequence, the resulting Internal Metric Timeline
shall always be identical.

---

## Consequences

Projection represents the first transition from observable temporal
events to the internal temporal representation of the ensemble.

Subsequent analytical stages operate exclusively on the Internal Metric
Timeline generated by this process.
