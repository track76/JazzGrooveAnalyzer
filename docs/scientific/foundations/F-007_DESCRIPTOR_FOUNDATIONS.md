# F-007 — Descriptor Foundations

## Status

Draft

## Purpose

This document formally defines the concept of Behaviour Descriptor.

A Behaviour Descriptor is the first quantitative representation
derived from Behaviour Representation.

Descriptors never modify the represented behaviour.

They only measure observable properties.

## Scientific Definition

A Behaviour Descriptor is a reproducible quantitative measurement
computed from a Behaviour Observation.

Descriptors are:

- deterministic;
- reproducible;
- observable-derived;
- independent from musical interpretation.

## Invariants

A Descriptor:

- never changes the represented behaviour;
- never introduces interpretation;
- never depends on future observations;
- is completely determined by its input.

## Architectural Position

Behaviour Observation
        ↓
Behaviour Profile
        ↓
Behaviour Descriptor
