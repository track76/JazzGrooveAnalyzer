# F-010 — Descriptor Relations

## Status

Draft

## Purpose

This document formally defines the concept of Descriptor Relation.

A Descriptor Relation is the first mathematical object produced
by Descriptor Algebra.

Relations do not introduce interpretation.

They formalize mathematically observable connections between
existing Behaviour Descriptors.

---

# Scientific Definition

A Descriptor Relation represents a deterministic mathematical
relationship between two or more BehaviourDescriptor objects.

A relation never creates new descriptors.

A relation never modifies existing descriptors.

---

# Input

DescriptorSet

---

# Output

DescriptorRelation*

---

# Principles

A Descriptor Relation:

- is deterministic;
- is immutable;
- preserves provenance;
- references only existing descriptors;
- contains no musical interpretation.

---

# Constraints

Relations are mathematical objects.

They are not statistical models.

They are not classifications.

They are not predictions.

They are not musical semantics.
