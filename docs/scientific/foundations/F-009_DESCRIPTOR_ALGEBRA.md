# F-009 — Descriptor Algebra

## Status

Draft

## Purpose

Descriptor Algebra defines how Descriptor Sets
are transformed into Analytical Structures.

## Input

DescriptorSet

## Output

AnalyticalStructure

## Principles

Descriptor Algebra:

- never modifies descriptors;
- never recomputes descriptors;
- only analyses descriptor relationships.

## Allowed Operations

- grouping
- ordering
- comparison
- aggregation
- correlation

## Forbidden Operations

- reinterpret Behaviour Representation;
- generate new observations;
- modify Descriptor values.
