# JGA Metric Context

Status: Draft

## Purpose

This document defines the scientific concept of Metric Context within the Jazz Groove Analyzer (JGA).

The Metric Context represents the first coherent temporal organization that emerges from observable musical evidence.

It does not represent musical interpretation, but the observable temporal structure from which semantic analysis may later be derived.

---

## Definition

The Metric Context is the temporal framework constructed from observable musical events.

It emerges progressively through the organization of temporal evidence rather than being imposed a priori.

The Metric Context is therefore an emergent scientific representation.

---

## Observable Metric Context

The Observable Metric Context is entirely constructed from measurable observations.

It may include:

- temporal regularities;
- recurrent pulse candidates;
- temporal alignment;
- periodic organization;
- collective rhythmic behaviour.

No musical semantics are introduced at this stage.

---

## Emergence of Metric Context

The Metric Context emerges progressively during the observation process.

Its construction depends on the accumulation of consistent observable temporal evidence.

As additional evidence becomes available, the Metric Context becomes increasingly stable.

---

## Relationship with Level A0

Level A0 determines when metric observation may legitimately begin.

Only after the ensemble reaches a sufficiently stable rhythmic organization can the Metric Context emerge.

---

## Relationship with Observation

The Metric Context is an output of the Observation Model.

It summarizes the temporal organization observed within the musical performance.

Observation therefore precedes the construction of the Metric Context.

---

## Relationship with TAC

The Temporal Alignment Context (TAC) describes the temporal relationships among observable events.

These relationships provide the structural information required for the emergence of the Metric Context.

TAC organizes temporal evidence.

The Metric Context represents the coherent temporal organization that results from that evidence.

---

## Relationship with the Domain Model

The Domain Model consumes the Metric Context through the Translation Layer.

Semantic entities such as Pulse, Metric Cluster and Beat Reference are never created directly from the audio signal.

They emerge only after the Metric Context has been scientifically established.

---

## Scientific Principles

The Metric Context satisfies the following principles:

- observation precedes interpretation;
- temporal organization precedes semantic representation;
- observable evidence remains independent from implementation;
- scientific models remain reproducible.

---

## Architectural Consequences

The Metric Context represents the scientific boundary between observation and semantic modelling.

It therefore becomes one of the principal inputs of the Translation Layer connecting the computational Core with the Domain Model.
