# BM-014 — Behaviour States

## Status

Scientific Foundation

## Purpose

This document formally defines Behaviour States.

Behaviour States are the elementary analytical
configurations of Behaviour Space.

Every Behaviour Trajectory is an ordered sequence
of Behaviour States.

## Scientific Principle

Observable rhythmic behaviour evolves through
successive analytical states.

Behaviour States constitute the fundamental
entities of Behaviour Dynamics.

## Definition

Let

B

be the Behaviour Space.

A Behaviour State

s ∈ B

is a complete analytical description of
observable behaviour at a given analytical instant.

## Input

Observable Behaviour

↓

Descriptor Algebra

↓

Behaviour Mathematics

## Transformation

State Identification

## Output

Behaviour State

## Interpretation

A Behaviour State represents a single point of
Behaviour Space.

It does not represent an event.

It does not represent a trajectory.

It represents an instantaneous analytical
configuration.

## Properties

Behaviour States

- belong to Behaviour Space;

- are uniquely identifiable within an analytical
context;

- may belong to Behaviour Regimes;

- may participate in Behaviour Trajectories;

- are implementation independent.

## Architectural Consistency

Behaviour States belong exclusively to
Behaviour Mathematics.

They never modify

- BehaviourDescriptors;

- DescriptorRelations;

- AnalyticalStructures.

## Scientific Consequences

Behaviour States establish the mathematical
foundation for

- Behaviour Dynamics;

- Behaviour Equilibria;

- Behaviour Attractors;

- Behaviour Geometry;

- future analytical state-space models.
