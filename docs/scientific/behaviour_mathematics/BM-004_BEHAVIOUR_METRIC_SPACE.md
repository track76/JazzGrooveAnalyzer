## Purpose

This document defines the Behaviour Metric Space.

mathematical environment in which Behaviour

A Behaviour Metric does not exist in isolation.

It induces a metric space over Behaviour Space.

Let

d : B × B → ℝ≥0

be a valid Behaviour Metric.

Then

(B,d)

is called the Behaviour Metric Space.
