# Jazz Groove Analyzer (JGA)

*A behaviour-based framework for the analysis of collective rhythmic interaction.*

---

## Overview

The **Jazz Groove Analyzer (JGA)** is a research project aimed at developing a computational framework for analysing rhythmic interaction within jazz ensembles.

Unlike conventional beat-tracking or onset-detection systems, the JGA does not assume that the metric reference is directly observable. Instead, it reconstructs the metric framework from the collective rhythmic behaviour of the ensemble.

The project combines concepts from music theory, rhythm analysis, computational modelling and explainable artificial intelligence to study how groove emerges through musical interaction.

---

# Vision

The JGA seeks to model musical timing as a collective cognitive phenomenon rather than as a sequence of isolated temporal events.

Its objective is not simply to measure timing accuracy, but to understand how musicians collaboratively construct, negotiate and transform metric organization during performance.

---

# Philosophy

> **The metric reference is not detected.**
>
> **It emerges from the collective rhythmic behaviour of the ensemble.**

This principle represents the conceptual foundation of the entire project.

**Research Stage**
